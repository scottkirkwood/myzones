My Zones
======

A simple utility to show some time zone information.

Links
-----
wxPython: http://www.wxpython.org/
pytz: http://pytz.sourceforge.net/

Install
-------

# python setup.py install

or

# easy_install.py mm2s5

Prerequesites
-------------

Requires Python 2.3 or greater 
Requires wxPython
Requires pytz
